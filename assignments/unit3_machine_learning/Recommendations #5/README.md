


## Dataset Description ratings.csv
|Variable Description | Variable Description | 
| ------------------- | --------------------- |
|userId | A unique Id represents every user|
|movieId | Unique ID representing a movie|
|rating | Represents the rating given by the user to the corresponding movie|
|timestamp | The time at which the rating was recorded |

## Dataset Description movies.csv
|Variable Description | Variable Description                                         | 
| ------------------- |--------------------------------------------------------------|
| movieId | A unique Id represents every movie                           | 
|title | Movie name which is represented by the corresponding movieId |
| genres | Represents the category of the movie                         | 
| timestamp | The time at which the rating was recorded                    |
